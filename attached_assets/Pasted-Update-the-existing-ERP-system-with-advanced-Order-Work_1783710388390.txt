Update the existing ERP system with advanced Order Workflow.

Do not rebuild existing modules.
Modify and connect the current workflow only.

New Order Classification:

When a new order is created:

Automatically classify order type:

1. Production Order
- Product requires manufacturing

2. Stock Order
- Product available in inventory


Production Order Workflow:

Orders
↓
Preparation
↓
Workshop Assignment
↓
Cutting
↓
Sewing
↓
Embroidery
↓
Quality Control
↓
Packaging
↓
Shipping
↓
Completed


System automatically creates:
- Production task
- Production record
- Link with Order ID
- Production timeline


Stock Order Workflow:

Orders
↓
Preparation
↓
Print Preparation Card
↓
Packaging
↓
Shipping
↓
Completed


No workshop or manufacturing steps required.
